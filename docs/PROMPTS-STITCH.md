# 1. 디자인 생성 (MCP)
    @stitch Digital Dice Roller 웹앱 만들어줘.

    ## 용도
    D&D, 보드게임용 주사위/동전 던지기 도구

    ## 사용자
    보드게이머, TRPG 플레이어

    ## 스타일
    - 무드: 판타지 RPG 느낌, 약간 레트로하고 게임스러운 분위기
    - 테마: 다크 브라운/골드 컬러, 양피지/가죽 텍스처 느낌
    - 폰트: 중세 판타지 느낌의 세리프체

    ## 레이아웃
    - 상단: 주사위 종류 선택 (d4, d6, d8, d10, d12, d20)
    - 중앙: 큰 주사위 결과 표시 영역
    - 하단: Roll 버튼과 히스토리

    ## 주요 요소
    - 주사위 아이콘들
    - 큰 Roll 버튼 (눈에 띄게)
    - 결과 숫자 크게 표시
    - 최근 굴림 히스토리 리스트
    - 동전 던지기 탭

    ## 참고
    Roll20 느낌, 던전앤드래곤 캐릭터 시트 스타일
